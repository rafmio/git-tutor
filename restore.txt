restore
ОЧЕНЬ ОПАСНАЯ КОМАНДА!
Никогда не использовать, если не знаешь точно для чего она.

С версии 2.23.0 Git - новая команда git restore. По сути это альтернатива
git reset

Допустим, мы изменили два файла, и случайно проиндексировали оба, хотя не хотели:
$ git add *

Для отмены индексации одного из нужных файлов:
$ git restore --staged <filename>
